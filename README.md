# ToDo List Web App



A simple flask todo list web app

### Requirements:

* flask
* flask_sqlalchemy

### Install:

- git clone git@github.com:superchilli/todolist-flask.git
- open terminal, `cd todolist-flask`
- `pip install flask`
- `pip install flask_sqlalchemy`
- `python todolist.py`
